# ZeroFoodWaste

## Description
ZeroFoodWaste is an app I created to help reduce food wastage, by providing users with recipes that have adjusted ingredient amounts to meet their caloric requirements 

## Built With 
- Python 
- Django
- React 
- REST APIs
- Spoonacular Recipe API
